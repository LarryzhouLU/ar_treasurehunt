# ar_treasurehunt
 ar寻宝游戏
 
 ## 游戏平台
 - android pad or phone
 - designed for ipad size
 
 ## 游戏玩法
 - 玩家需要通过线索扫描正确的Vumark，并答对相应的问题，最终获得游戏的胜利。
 - 游戏限时30分钟，时间结束，任务失败
